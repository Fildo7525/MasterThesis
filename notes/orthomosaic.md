### Orthomosaic processing in Metashape

1. Load images for multicamera system
2. 0 on the keyboard resets the view
3. check if camera is selected
4. Chcek if the 3D looks about right in shape
5. If not aligned you can force the alignment
6. Calibrate reflectance in the top tools - use sun sensor
7. Make sure in the photos tab that the one that shows up is RGB, others should be BLACK
8. Change the coordinates system to UTM 32N
9. Select chunks basedon number of datasets
10. DO not sync it with nextcloud. do it locally in a folder
11. Repeat the steps for all the datasets (chunks)
12. Name the chunks based on data and place andd dataset name
13. Go to batch processing /align photos
14. Go to high settings and the rest is default
15. Do processing for all chunks and save each step
16. After it is one for all chunks, do verification
17. Does it look good, does camera callibration make sense?
18. Optimised camera (star button at left top) - fit additional corrections or something like that
19. Save 
20. First without pointcloud - go to batch prcossing and switch of pointcloud (buiild DM)
21. Create orthomosaic (build orthomosaic) - check your units
22. Evaluate them visually
23. Hand processing for the dataset
24. Duplicate chunk and create a point cloud
25. Build orthomosaic based on point cloud
26. Compare orthomosaics w and withut point cloud
